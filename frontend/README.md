# Frontend (React)

Dieses Verzeichnis enthält das Frontend der Sauna-Aufguss-Koordinations-App.

## Stack & Abhängigkeiten
- Node.js (empfohlen: v20+)
- React
- TypeScript
- FullCalendar

## Setup
1. Node-Module installieren:
   ```bash
   npm install
   ```
2. Entwicklungsserver starten:
   ```bash
   npm start
   ```
